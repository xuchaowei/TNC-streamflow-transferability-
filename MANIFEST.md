# Package Manifest

This English GitHub package contains:

- MATLAB scripts for source-basin T-N-C streamflow forecasting and TVFEMD decomposition.
- MATLAB scripts for NRBO-based global transferability experiments.
- R scripts for plotting regional performance distributions.
- Compact example CSV files showing the expected public-dataset input format.
- Compact process-model benchmark files for XinAnJiang and SWAT/SWAT-CUP calibration transparency.

The following materials were intentionally excluded:

- Large SWAT output databases and daily output tables.
- GIS rasters, model databases, executable dependencies, and third-party binaries.
- Thousands of public basin time-series files that should be downloaded from their original data providers.
- Local machine-specific paths and non-English setup notes.

This package is designed for code transparency and journal review. Large data and model-output reconstruction should follow `DATA_AVAILABILITY.md`.
