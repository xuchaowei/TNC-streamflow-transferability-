# Global transferability and NRBO scripts

This folder contains MATLAB scripts used for regional/global transfer experiments and R scripts used to generate violin/raincloud-style diagnostic plots.

## Structure

- `regions/`: MATLAB scripts grouped by global transfer region.
- `example_data/`: a small number of representative basin CSV files for checking script format.
- `plotting/`: R scripts for distribution and significance-style figures.

## Important data note

The complete global daily basin time-series are not duplicated here because they include thousands of public-data files and are better obtained from their original sources. See `../DATA_AVAILABILITY.md` for dataset links and recommended placement.
