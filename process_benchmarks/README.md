# Process-driven benchmark models

This folder contains files retained for transparency of the process-driven benchmark comparison.

## Included components

- `xinanjiang_models/matlab/`: MATLAB code for XinAnJiang-type benchmark simulations and metrics.
- `swat_swatcup/`: SWAT-CUP parameter, observation, calibration and compact output files.

## Excluded components

Large SWAT output files, GIS rasters/databases, ArcGIS/QSWAT/SWAT executable dependencies, and third-party binaries were not copied into this GitHub package. They are not source code and are not suitable for normal GitHub storage. See `../manifest/excluded_files.csv`.
